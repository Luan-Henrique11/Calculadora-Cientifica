----------- # Calculadora-Cientifica # ----------

Uma calculadora científica feita em Python + Kivy, com interface moderna, funções científicas, parênteses inteligentes, visor duplo (expressão + resultado) e histórico rolável das últimas operações.

A calculadora contem as seguintes funcionalidades:
* Operações básicas: + - * / %
* Funções científicas: sin, cos, tan (em graus), log (base 10), ln (natural), sqrt (raiz), abs (valor absoluto), Potência x² e xʸ, Constantes pi
* Parênteses inteligentes ( )
* Inverter sinal ±
* Botões especiais: limpar C, apagar ⌫, calcular =
* Histórico com até 20 cálculos, com rolagem (ScrollView)
* Formatação automática do resultado: remove .0 quando o resultado é inteiro e limita para até 10 algarismos significativos

O projeto utiliza: Python 3; Kivy; Biblioteca math

Projeto livre para estudo e portfólio.
